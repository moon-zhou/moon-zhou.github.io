github新增代码库实操

2019-06-30

git创建新的代码库

  1. 登录github
  2. 找到”new repository”

本地操作

  1. 创建git目录：进入目录，git init
  2. 将所有文件添加到本地工作区：git add .
  3. 提交到本地库：git commit -m 提交说明
  4. 添加本地和远程仓库的关联：git remote add localRepositoryName remoteRepositoryUrl【示例：git remote add my-project-first https://github.com/moon-zhou/VueDemo.git】
  5. 第一次不需要pull，直接push，需要设置upstream，后面就直接push，git push –set-upstream my-project-first master

后悔药系列

  1. 如果github上传时，设置的提交人名称有误
    1. 删除上一次提交，如果就一次提交，或者只有一个作者，因为换机器导致提交用户错误的，直接删除代码库，重新新建github代码库
    2. 本地删除git管理文件：rm -rf .git
    3. 修改本地的git用户名：git config –global user.name，==详情参考Git学习实操==
    4. 重新参考”本地操作”，重新走一遍
